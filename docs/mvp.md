# MVP — Chess AI Platform

## 1. Project Name
Chess AI Platform

## 2. Project Description
Web platform for playing chess and analyzing games using artificial intelligence. Users can play chess on an interactive board, save games, import and export PGN/FEN, and analyze positions using the Stockfish engine. In future versions, the platform may include its own AI engine for deeper analysis and training.

## 3. Project Goal
To build a working web platform where users can play chess, save their games, and analyze positions and completed matches using Stockfish.

## 4. Target Users
- Beginner chess players
- Intermediate and advanced players
- Users who want to analyze their games
- Students and coaches

## 5. Core MVP Features
- Interactive chess board
- Local chess game
- Play against AI
- Move history
- PGN/FEN import and export
- Position analysis
- Game saving
- User authentication

## 6. Not Included in MVP
- Tournaments
- Chat system
- Anti-cheat
- Mobile application
- Custom AI engine
- Elo rating system

## 7. Technologies
### Frontend
- Next.js
- TypeScript

### Backend
- FastAPI

### Database
- PostgreSQL

### Engine
- Stockfish
